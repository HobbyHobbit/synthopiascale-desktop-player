## Getting Started

### Installation

1. Download the installer from [synthopiascale.com](https://synthopiascale.com)
2. Run `SynthopiaScale Desktop Player-Setup-x.x.x.exe`
3. Follow the installation wizard
4. Launch the application from your Start Menu or Desktop

### First Launch

On first launch, the player opens in **Quality Mode** with the full interface visible. You can switch between modes using the buttons in the top-right corner.

## Library Management

### Views

The library sidebar has multiple views:

1. **Queue** - Current playback queue (drag to reorder)
2. **Library** - All imported tracks
3. **Playlists** - Your saved playlists

### Queue Management

- **Drag & Drop** - Reorder tracks in the queue
- **Right-click** - Context menu with options:
  - Play
  - Remove from queue
  - Show in folder

### Creating Playlists

1. Go to **Playlists** view
2. Click **+ New Playlist**
3. Enter a name
4. Add tracks from your library

### Import/Export Playlists

- **Import** - Supports M3U playlist files
- **Export** - Save playlists as M3U for backup or sharing

## Equalizer

### Opening the EQ

Press **E** or click the EQ button (ðŸŽšï¸) in the top controls.

### 10-Band Frequencies

| Band | Frequency |
|------|-----------|
| 1 | 32 Hz (Sub Bass) |
| 2 | 64 Hz (Bass) |
| 3 | 125 Hz (Low Mid) |
| 4 | 250 Hz (Mid) |
| 5 | 500 Hz (Mid) |
| 6 | 1 kHz (Upper Mid) |
| 7 | 2 kHz (Presence) |
| 8 | 4 kHz (Brilliance) |
| 9 | 8 kHz (Air) |
| 10 | 16 kHz (Sparkle) |

### Presets

Built-in presets include:
- **Flat** - No EQ adjustment
- **Bass Boost** - Enhanced low frequencies
- **Treble Boost** - Enhanced high frequencies
- **Club** - Powerful bass for dance music
- **Hi-Fi** - Balanced audiophile curve
- **Vocal** - Emphasizes vocal frequencies
- **Electronic** - Optimized for EDM
- **Rock** - Classic rock EQ curve
- **Jazz** - Warm, smooth tone
- **Classical** - Natural, dynamic range

### Anti-Clipping

The EQ features automatic **Pre-Gain** adjustment to prevent clipping when boosting frequencies. This ensures clean audio output even with aggressive EQ settings.

## Troubleshooting

### Audio Not Playing

1. Check if a track is loaded (visible in player)
2. Ensure volume is not muted (M key)
3. Check system volume mixer
4. Try a different audio file

### Visualizer Not Showing

1. Check Settings > Visual Effects > Audio Visualization is ON
2. Ensure Intensity is above -90%
3. Try switching visualizer type

### Performance Issues

1. Switch to **Performance Mode** (âš¡)
2. Disable Background Particles
3. Disable Particle Hover Effect
4. Lower Intensity setting

### Files Not Loading

Supported formats: MP3, WAV, FLAC, OGG, M4A, AAC, OPUS, WEBM

If files still won't load:
1. Check file isn't corrupted
2. Try re-encoding the file
3. Ensure file path doesn't contain special characters
